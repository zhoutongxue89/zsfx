1. 下载与安装

下载地址

访问 VS Code官网，下载对应系统的版本：

  • Windows：下载 .exe 安装包
  • macOS：下载 .zip 包或通过 Homebrew 安装
  • Linux：下载 .deb 或 .rpm 包
💡 提示：推荐下载 User Installer 版本，安装时选择"为所有用户安装"可以获得更好的体验。

初始设置

  1. 安装完成后打开VS Code
  2. 首次启动时会让你选择主题，建议选择 Dark+ 或 One Dark Pro
  3. 安装中文语言包：按 Ctrl+Shift+P，输入 Configure Display Language，选择 中文(简体)

2. 安装Python扩展

VS Code对Python的支持主要依靠微软官方的Python扩展：

安装步骤

  1. 按 Ctrl+Shift+X 打开扩展面板
  2. 在搜索框中输入 Python
  3. 找到微软官方的 Python 扩展（作者：Microsoft）
  4. 点击 安装
✅ Python扩展功能：
  • 语法高亮和代码补全
  • 代码调试支持
  • 代码格式化
  • 单元测试支持
  • 虚拟环境管理
  • Jupyter Notebook支持

3. 基础配置

打开设置

按 Ctrl+, 打开设置界面，或通过菜单：文件 → 首选项 → 设置

推荐配置项

{
    // 自动保存
    "files.autoSave": "afterDelay",
    
    // 字体大小
    "editor.fontSize": 14,
    
    // 字体家族
    "editor.fontFamily": "Consolas, 'Courier New', monospace",
    
    // 行号显示
    "editor.lineNumbers": "on",
    
    // 缩进指南
    "editor.renderLineHighlight": "all",
    
    // 空格代替Tab
    "editor.insertSpaces": true,
    "editor.tabSize": 4,
    
    // 自动格式化
    "editor.formatOnSave": true,
    
    // Python路径（根据你的安装位置调整）
    "python.defaultInterpreterPath": "python",
    
    // 文件排除
    "files.exclude": {
        "**/__pycache__": true,
        "**/*.pyc": true,
        "**/.git": true
    }
}

settings.json完整配置

按 Ctrl+Shift+P，输入 Preferences: Open Settings (JSON)，将上面的配置粘贴进去。

6. 调试配置

创建launch.json

  1. 按 F5 或点击左侧调试图标
  2. 选择 Python File 作为调试配置
  3. VS Code会自动创建 .vscode/launch.json 文件

常用调试功能

  • 断点：点击代码行号左侧设置断点
  • 步进：F10单步跳过，F11单步进入
  • 变量：在调试控制台查看变量值
  • 监视：添加表达式监视
{
    "version": "0.2.0",
    "configurations": [
        {
            "name": "Python: Current File",
            "type": "python",
            "request": "launch",
            "program": "${file}",
            "console": "integratedTerminal"
        },
        {
            "name": "Python: Module",
            "type": "python",
            "request": "launch",
            "module": "${input:moduleName}",
            "console": "integratedTerminal"
        }
    ],
    "inputs": [
        {
            "id": "moduleName",
            "type": "promptString",
            "description": "输入模块名"
        }
    ]
}

7. 实用技巧

多光标编辑

Ctrl + Alt + Down   # 在下方添加光标
Ctrl + Alt + Up     # 在上方添加光标
Ctrl + Shift + L    # 选择所有相同内容同时编辑

代码片段

创建自定义代码片段：按 Ctrl+Shift+P，输入 Configure User Snippets，选择 python.json

{
    "Print to console": {
        "prefix": "print",
        "body": [
            "print(f'$1: $\{$1\}')"
        ],
        "description": "格式化打印"
    },
    "Main function": {
        "prefix": "main",
        "body": [
            "if __name__ == '__main__':",
            "    $1"
        ],
        "description": "主函数模板"
    }
}

终端集成

按 Ctrl+` 打开集成终端，可以直接运行Python代码：

python file.py          # 运行Python文件
python -m pip install   # 安装包

集成Git Bash (Windows)

在设置中添加：

"terminal.integrated.profiles.windows": {
    "Git Bash": {
        "source": "Git Bash",
        "path": ""
    }
},
"terminal.integrated.defaultProfile.windows": "Git Bash"
