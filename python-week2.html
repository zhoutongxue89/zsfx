Day 1: 数据类型初识

什么是数据类型？

数据类型是编程语言中用于区分不同种类数据的基本概念。就像现实生活中有不同的物品分类，程序中的数据也有不同的类型。

Python常用数据类型一览

类型名中文名示例说明
int整数42, -7没有小数点的数字
float浮点数3.14带小数点的数字
str字符串"Hello"文本数据
bool布尔值True, False真/假
NoneType空值None表示"什么都没有"

使用type()查看数据类型

# 基本用法
print(type(42))      # <class 'int'>
print(type(3.14))     # <class 'float'>
print(type("Hello"))  # <class 'str'>
print(type(True))     # <class 'bool'>
<class 'int'> <class 'float'> <class 'str'> <class 'bool'>

使用isinstance()检查类型

x = 42
print(isinstance(x, int))        # True
print(isinstance(x, str))        # False
print(isinstance(x, (int, float)))  # True (检查多个类型)

Day 2: 数字类型

整数(int)

整数是没有小数部分的数字，可以是正数、负数或零。Python中的整数没有大小限制。

a = 42           # 正整数
b = -7           # 负整数
million = 1_000_000  # 下划线分隔(3.6+)
print(million)    # 1000000

浮点数(float)

浮点数是带小数部分的数字，用于表示实数。

pi = 3.14159
# 科学计数法
avogadro = 6.02e23  # 6.02 × 10²³
electron = 1.6e-19 # 1.6 × 10⁻¹⁹

算术运算符

运算符名称示例结果
+加法10 + 313
-减法10 - 37
*乘法10 * 330
/除法10 / 33.333...
//整除10 // 33
%取模10 % 31
**幂运算10 ** 31000
 
a, b = 17, 5
print(f"{a} + {b} = {a + b}")   # 22
print(f"{a} // {b} = {a // b}") # 3
print(f"{a} % {b} = {a % b}")   # 2
print(f"{a} ** {b} = {a ** b}") # 1419857

赋值运算符

x = 10
x += 5   # x = x + 5  → 15
x *= 2   # x = x * 2  → 30
x /= 4   # x = x / 4  → 7.5
注意：浮点数精度问题

0.1 + 0.2 的结果是 0.30000000000000004，不是精确的0.3！这是计算机表示浮点数的固有问题。

Day 3: 字符串基础

字符串的定义

字符串是用引号包裹的文本数据。Python中可以用单引号 ' ' 或双引号 " " 来定义。

s1 = 'Hello, World!'        # 单引号
s2 = "Python is fun!"       # 双引号
s3 = '''多行
字符串'''                    # 三引号

转义字符

转义字符含义
\\反斜杠
\'单引号
\"双引号
\n换行
\tTab
 
print("第一行\n第二行")    # 换行输出
print("C:\\Users\\Admin") # C:\Users\Admin
# 原始字符串
print(r"C:\Users\Admin") # C:\Users\Admin

字符串索引

字符串是由字符组成的序列，每个字符都有一个索引（从0开始）。

s = "Python"
#  P    y    t    h    o    n
#  0    1    2    3    4    5    (正向)
# -6   -5   -4   -3   -2   -1    (负向)

print(s[0])    # P
print(s[-1])   # n
print(s[0:3])   # Pyt (索引0-2)
print(s[::-1])]) # nohtyP (反转)
切片记忆技巧

切片遵循"左闭右开"原则：s[0:3] 表示索引0,1,2，共3个字符。

Day 4: 字符串操作

字符串拼接

# 方法1: + 拼接
print("Hello" + ", " + "World")   # Hello, World

# 方法2: * 重复
print("-" * 20)                # --------------------

# 方法3: join() 连接列表
words = ["Python", "is", "awesome"]
print(" ".join(words))           # Python is awesome

大小写转换

方法作用示例
.upper()全部大写"hello".upper() → "HELLO"
.lower()全部小写"HELLO".lower() → "hello"
.capitalize()首字母大写"hello".capitalize() → "Hello"
.title()每个单词首字母大写"hello world".title() → "Hello World"

查找与判断方法

s = "Hello, Hello, Hello!"

# 查找
print(s.find("Hello"))    # 0 (第一次位置)
print(s.rfind("Hello"))   # 14 (最后位置)
print(s.count("Hello"))   # 3 (出现次数)

# 判断
print(s.startswith("Hello"))  # True
print(s.endswith("!"))      # True
print("123".isdigit())     # True

替换、分割、合并

# replace - 替换
print("Hello".replace("l", "L"))  # HeLLo

# split - 分割
print("a,b,c".split(","))     # ['a', 'b', 'c']

# join - 合并
print("-".join(["2024","03","22"]))  # 2024-03-22

# strip - 去除空白
print("  hi  ".strip())      # hi

Day 5: f-string格式化

f-string是Python 3.6+引入的最推荐的字符串格式化方法，以f或F开头，在字符串中使用{}包裹变量或表达式。

f-string优势：语法简洁、可读性强、支持表达式、性能最优

基本用法

name, age = "小明", 18
print(f"我叫{name}，今年{age}岁")  # 我叫小明，今年18岁

# 可以在{}中使用表达式
print(f"{10} + {3} = {10 + 3}")    # 10 + 3 = 13
print(f"{'hello'.upper()}")        # HELLO

格式化数字

import math
pi = math.pi

# 保留小数位
print(f"π ≈ {pi:.2f}")    # π ≈ 3.14
print(f"{0.756:.1%}")     # 75.6%

# 数字前导零
print(f"{42:05d}")       # 00042

# 千位分隔符
print(f"{1234567:,}")     # 1,234,567

# 进制转换
print(f"{255:#b}")       # 0b11111111
print(f"{255:x}")       # ff

对齐与填充

s = "Hi"
print(f"|{s:<10}|")   # |Hi        | 左对齐
print(f"|{s:>10}|")   # |        Hi| 右对齐
print(f"|{s:^10}|")   # |    Hi    | 居中
print(f"|{s:-^10}|")  # |----Hi----| 用-填充

打印花括号

print(f"{{}}")          # {}
print(f"{{1}} + {{2}} = 3")  # {1} + {2} = 3

Day 6: 布尔值与None

布尔值(bool)

布尔值是表示真(True)和假(False)的数据类型，用于条件判断和逻辑运算。

print(True)   # True
print(False)  # False
print(True == 1)   # True (True等于1)
print(False == 0)  # True (False等于0)

哪些值相当于False？

值说明
False布尔值False
None空值
0, 0.0数字零
""空字符串
[], {}, ()空容器

比较运算符

print(10 == 10)   # True
print(5 > 3)     # True
print(1 < 2 < 3)   # True (链式比较)

逻辑运算符

运算符含义说明
and逻辑与两边都为True才为True
or逻辑或任一边为True就为True
not逻辑非取反
 
print(True and False)   # False
print(True or False)    # True
print(not True)       # False

# 短路特性
name = ""
print(name or "匿名")    # 匿名 (空字符串为False)

None类型

None 表示空、无、什么都没有，它不等于0、空字符串或空列表。

x = None
print(x is None)     # True (推荐用is)
print(x == None)     # True (也可以，但不推荐)

# 处理可选参数
def greet(name=None):
    if name is None:
        print("你好！")
    else:
        print(f"你好，{name}！")
is 与 == 的区别

== 比较值是否相等，is 比较是否是同一个对象（内存地址相同）。

Day 7: 类型转换

Python遵循"显式优于隐式"的原则，类型转换必须显式调用转换函数。

转换为整数：int()

print(int("123"))         # 123
print(int(3.7))           # 3 (截断，不是四舍五入)
print(int("1010", 2))      # 10 (二进制转十进制)
print(int("FF", 16))      # 255 (十六进制)

转换为浮点数：float()

print(float("3.14"))     # 3.14
print(float("123"))      # 123.0
print(float(42))        # 42.0

转换为字符串：str()

print(str(123))         # "123"
print(str(3.14))        # "3.14"
print(str(True))        # "True"

转换为布尔值：bool()

print(bool(0))      # False
print(bool(123))     # True
print(bool(""))     # False
print(bool("0"))     # True ("0"是非空字符串)

实战：用户输入的类型转换

# input()返回的永远是字符串
age = int(input("请输入年龄: "))  # 输入"25" → 25
height = float(input("请输入身高: "))  # 输入"1.75" → 1.75
confirm = input("确认?(yes/no): ").lower() == "yes"
print(f"年龄:{age}, 身高:{height}, 确认:{confirm}")
常见错误 
# int("3.14") 会报错！
int(float("3.14"))  # 正确：先转float再转int

# 字符串和数字不能直接相加
# "年龄: " + 18  # 报错！
print("年龄: " + str(18))  # 正确
