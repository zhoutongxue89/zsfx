字典是什么？

字典是一个键值对（key-value）的集合，用花括号 {} 表示。就像真实的字典：用单词（键）查含义（值）。

类比：字典就像手机通讯录 —— 用名字（键）找到电话号码（值），查找速度极快！
⚡

极速查找

无论数据多大，查找时间几乎不变（哈希表）

🔑

键值对

每个数据都是 key: value 的形式

📋

有序（3.7+）

Python 3.7起，字典保持插入顺序

🔒

键唯一

键不能重复，值可以重复

创建字典与访问

# 直接创建
person = {
    "name": "周同学",
    "age": 28,
    "city": "北京",
    "hobbies": ["Python", "摄影", "旅行"]
}

# 用 dict() 创建
config = dict(host="localhost", port=3306, db="mydb")

# 用键访问值
print(person["name"])    # 周同学
print(person["age"])     # 28
print(person["hobbies"]) # ['Python', '摄影', '旅行']

# 用 get() 安全访问（键不存在不报错）
print(person.get("email"))              # None
print(person.get("email", "未填写"))   # 未填写（默认值）
print(person.get("name", "未知"))     # 周同学
⚠️ 注意：用 dict[key] 访问不存在的键会报 KeyError，建议用 dict.get(key) 更安全！

增删改查操作

person = {"name": "周同学", "age": 28}

# 增：直接赋值（键不存在则新增）
person["city"] = "北京"
person["email"] = "zhou@example.com"
print(person)

# 改：直接赋值（键存在则修改）
person["age"] = 29
print(person["age"])  # 29

# 删：del 关键字
del person["email"]

# 删：pop() 方法（返回被删除的值）
age = person.pop("age")
print(f"删除的age值：{age}")

# 查：in 判断键是否存在
print("name" in person)   # True
print("age" in person)    # False（已被删除）

# 批量更新
person.update({"age": 29, "phone": "138xxxx"})
print(person)

字典常用方法（12个）

方法说明示例
d.get(key, default)安全获取值，键不存在返回默认值d.get("age", 0)
d.keys()返回所有键的视图list(d.keys())
d.values()返回所有值的视图list(d.values())
d.items()返回所有键值对的视图list(d.items())
d.update(other)用other字典更新d（批量添加/修改）d.update({"a":1})
d.pop(key)删除键并返回值val = d.pop("x")
d.popitem()删除并返回最后插入的键值对k,v = d.popitem()
d.setdefault(k, v)键不存在时设置默认值d.setdefault("count", 0)
d.clear()清空字典d.clear()
d.copy()浅复制d2 = d.copy()
len(d)键值对数量len(d)
key in d判断键是否存在"name" in d
# setdefault 非常有用：统计词频
text = "apple banana apple cherry banana apple"
word_count = {}
for word in text.split():
    word_count.setdefault(word, 0)  # 不存在时初始化为0
    word_count[word] += 1
print(word_count)
# {'apple': 3, 'banana': 2, 'cherry': 1}

# 更简洁：用 get
word_count2 = {}
for word in text.split():
    word_count2[word] = word_count2.get(word, 0) + 1
print(word_count2)

字典遍历

scores = {"小明": 85, "小红": 92, "小李": 78}

# 遍历键（默认）
for name in scores:
    print(name)  # 小明、小红、小李

# 遍历值
for score in scores.values():
    print(score)  # 85、92、78

# 遍历键值对（最常用！）
for name, score in scores.items():
    grade = "优" if score >= 90 else "良"
    print(f"{name}：{score}分（{grade}）")

# 按值排序遍历
for name, score in sorted(scores.items(), key=lambda x: x[1], reverse=True):
    print(f"{name}：{score}")
# 小红92、小明85、小李78

字典推导式

# 基础：创建平方字典
squares = {x: x**2 for x in range(1, 6)}
print(squares)  # {1: 1, 2: 4, 3: 9, 4: 16, 5: 25}

# 反转字典（键值互换）
original = {"a": 1, "b": 2, "c": 3}
reversed_dict = {v: k for k, v in original.items()}
print(reversed_dict)  # {1: 'a', 2: 'b', 3: 'c'}

# 带条件过滤（只保留分数≥80的）
scores = {"小明": 85, "小红": 92, "小李": 55, "小张": 78}
passed = {name: score for name, score in scores.items() if score >= 80}
print(passed)  # {'小明': 85, '小红': 92}

# 从两个列表创建字典
keys = ["name", "age", "city"]
values = ["周同学", 28, "北京"]
person = dict(zip(keys, values))
print(person)

嵌套字典

字典的值可以是另一个字典，形成树形结构。

# 学生信息数据库
database = {
    "S001": {
        "name": "小明",
        "scores": {"语文": 85, "数学": 92, "英语": 88},
        "grade": "三年级"
    },
    "S002": {
        "name": "小红",
        "scores": {"语文": 95, "数学": 78, "英语": 96},
        "grade": "三年级"
    }
}

# 访问嵌套数据
print(database["S001"]["name"])              # 小明
print(database["S001"]["scores"]["数学"])  # 92

# 遍历嵌套字典
for sid, info in database.items():
    avg = sum(info["scores"].values()) / len(info["scores"])
    print(f"{sid} {info['name']}：平均分 {avg:.1f}")

# 修改嵌套数据
database["S001"]["scores"]["语文"] = 90
database["S001"]["contact"] = {"phone": "138xxxx"}  # 添加新键

集合（Set）详解

集合是一个无序、不重复的元素集合，用花括号 {} 表示（但和字典不同，没有键值对）。

🚫

自动去重

重复元素自动删除，完美用于去重

⚡

查找极快

判断元素是否存在，比列表快很多

🔢

集合运算

支持交集、并集、差集等数学运算

# 创建集合
fruits = {"苹果", "香蕉", "橙子"}
numbers = {1, 2, 3, 4, 5}
empty_set = set()  # 注意：{} 创建的是空字典！

# 自动去重！
duplicates = [1, 2, 2, 3, 3, 3, 4]
unique = set(duplicates)
print(unique)  # {1, 2, 3, 4}

# 常用操作
s = {1, 2, 3}
s.add(4)           # 添加元素
s.remove(2)        # 删除元素（不存在会报错）
s.discard(10)      # 安全删除（不存在不报错）
print(3 in s)       # True（成员判断）
print(len(s))       # 集合大小

# 实际用途：列表去重
names = ["小明", "小红", "小明", "小李", "小红"]
unique_names = list(set(names))
print(unique_names)  # ['小明', '小红', '小李']（顺序可能不同）

集合运算（数学味十足）

A = {1, 2, 3, 4, 5}
B = {4, 5, 6, 7, 8}

# 交集：两个集合共同拥有的
print(A & B)          # {4, 5}
print(A.intersection(B))  # 同上

# 并集：两个集合合在一起（去重）
print(A | B)          # {1, 2, 3, 4, 5, 6, 7, 8}
print(A.union(B))         # 同上

# 差集：A有但B没有
print(A - B)          # {1, 2, 3}
print(A.difference(B))    # 同上

# 对称差集：只属于A或只属于B（不包括共有）
print(A ^ B)          # {1, 2, 3, 6, 7, 8}
print(A.symmetric_difference(B))  # 同上

# 子集判断
C = {1, 2}
print(C.issubset(A))   # True（C是A的子集）
print(A.issuperset(C)) # True（A包含C）
print(A.isdisjoint(B)) # False（有交集）

🎯 实际应用：找出两个班级共同选修的课程

class_A_courses = {"Python", "数学", "英语", "历史"}
class_B_courses = {"Python", "物理", "英语", "化学"}

common = class_A_courses & class_B_courses
only_A = class_A_courses - class_B_courses
only_B = class_B_courses - class_A_courses
all_courses = class_A_courses | class_B_courses

print(f"共同课程：{common}")     # {'Python', '英语'}
print(f"只有A：{only_A}")       # {'数学', '历史'}
print(f"只有B：{only_B}")       # {'物理', '化学'}
print(f"所有课程：{all_courses}")

综合实战

🎯 实战：词频统计 + 排行榜

from collections import Counter

text = """Python是一种简洁优雅的编程语言。
Python非常易学，Python社区也很活跃。
学习Python让你的编程之路更加顺畅。"""

# 简单词频统计
words = text.replace("。", "").replace("，", "").replace("。\n", "").split()
counter = {}
for word in words:
    counter[word] = counter.get(word, 0) + 1

print("=== 词频统计 ===")
for word, count in sorted(counter.items(), key=lambda x: x[1], reverse=True):
    bar = "▓" * count
    print(f"{word:8s} {count:2d}次 {bar}")

# 用Counter更简洁
word_counter = Counter(words)
print("\n最常见的3个词：", word_counter.most_common(3))
知识点核心要点掌握度
字典创建{key: value} 或 dict()必须掌握
d.get(key, default)安全取值，避免KeyError必须掌握
d.items()遍历键值对必须掌握
字典推导式{k:v for k,v in ...}推荐掌握
嵌套字典字典的值也是字典推荐掌握
集合去重list(set(lst))必须掌握
集合运算& | - ^推荐掌握
