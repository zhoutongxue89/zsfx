1. PyCharm简介

两个版本

版本 价格 特点
PyCharm Community 免费 适合Python开发，基础功能完善
PyCharm Professional 付费/学生免费 Web开发、数据科学、Django/Flask支持

主要功能

  • 智能代码补全 - 基于语义的代码补全
  • 代码重构 - 安全可靠的代码重构工具
  • 调试器 - 强大的调试功能
  • 版本控制 - 内置Git/ SVN支持
  • 远程开发 - 支持SSH、Docker等远程环境
  • 数据库工具 - 数据库管理和SQL支持

2. 下载与安装

下载地址

访问 JetBrains官网

Windows安装

  1. 下载 .exe 安装包
  2. 运行安装程序
  3. 建议勾选：
    • Create Desktop Shortcut
    • Create Associations (.py)
    • Add launchers dir to PATH
  4. 安装完成，首次启动会引导配置

学生免费申请

  1. 访问 学生申请页面
  2. 使用学校邮箱注册或上传学生证明
  3. 审核通过后获得一年免费使用权
✅ 提示：如果你是学生，使用学校邮箱（.edu结尾）可以免费申请PyCharm Professional！

3. 创建项目

新建项目

  1. 启动PyCharm，点击 New Project
  2. 设置项目位置和名称
  3. 选择Python解释器：
    • New virtualenv - 创建新的虚拟环境
    • Existing interpreter - 使用已安装的Python
    • Conda Environment - 使用Conda环境
  4. 点击 Create 创建项目

项目结构

myproject/
├── .idea/              # PyCharm配置（不要手动修改）
├── venv/               # 虚拟环境
├── src/                # 源代码目录
│   └── main.py         # 主程序文件
├── tests/              # 测试代码
├── requirements.txt    # 依赖列表
└── README.md           # 项目说明

运行代码

# 方式1：右键运行
# 在编辑区右键点击 → Run 'main'

# 方式2：使用快捷键
Shift + F10

# 方式3：使用工具栏
点击右上角的绿色三角形按钮

4. 基础配置

打开设置

Windows: File → Settings 或 Ctrl+Alt+S

macOS: PyCharm → Preferences 或 Cmd+,

外观设置

# 主题设置
Settings → Appearance
- Theme: 选择深色/浅色主题
- Font: 设置编辑器字体

# 编辑器字体
Settings → Editor → Font
- Font: JetBrains Mono 或 Consolas
- Size: 14
- Line height: 1.2

Python解释器设置

Settings → Project → Python Interpreter

# 添加新的解释器
1. 点击齿轮图标 → Add
2. 选择解释器类型：
   - System Interpreter：系统Python
   - Virtualenv Environment：虚拟环境
   - Conda Environment：Conda环境

# 安装包
1. 在包列表下方点击 +
2. 搜索包名并安装

代码风格设置

Settings → Editor → Code Style → Python

# 缩进
- Tab size: 4
- Indent: 4
- Continuation indent: 8

# 行长度
- Hard wrap at: 120

# 导入排序
Settings → Editor → Code Style → Python → Imports
- Sort imports by length: ✓
- Force rearrange: ✓

自动保存和备份

Settings → Appearance & Behavior → System Settings
- Save files automatically: ✓
- Save files on frame deactivation: ✓

Settings → Appearance & Behavior → System Settings → Backup
- Automatically save backups: ✓

6. 调试功能

设置断点

  1. 点击代码行号左侧的空白区域
  2. 或使用快捷键 Ctrl + F8

断点类型

  • 普通断点 - 暂停执行
  • 条件断点 - 右键断点 → 设置条件（如 i > 10）
  • 日志断点 - 打印日志而不暂停
  • 异常断点 - 在出错误时暂停

调试面板

# 调试时的面板
- Frames: 调用栈
- Variables: 当前变量值
- Watches: 自定义监视表达式
- Console: Python交互控制台

# 调试时执行代码
1. 在Debug Console中
2. 可以输入表达式查看结果
3. 可以修改变量值

Attach to Process

如果需要调试正在运行的Python进程：

Run → Attach to Process
# 或 Ctrl + Alt + F5

7. Git集成

启用版本控制

  1. VCS → Import into Version Control → Create Git Repository
  2. 选择项目目录

Git面板

# 打开Git面板
View → Tool Windows → Git
# 或 Alt + 9

# 显示内容
- Log: 提交历史
- Local Changes: 未提交的更改
- Console: Git命令输出

常用操作

# 提交
1. VCS → Commit（或 Ctrl + K）
2. 选择要提交的文件
3. 填写提交信息
4. 点击 Commit

# 推送
VCS → Git → Push
或 Ctrl + Shift + K

# 拉取
VCS → Git → Pull

# 分支
VCS → Git → Branches
或右下角的分支图标

解决冲突

  1. 合并时出现冲突，点击 Merge
  2. 左侧是你的版本，中间是结果，右侧是合并版本
  3. 点击 ← 或 → 接受对应版本
  4. 或手动编辑中间区域
  5. 点击 Apply 完成
