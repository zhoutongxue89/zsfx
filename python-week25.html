2️⃣ Flask框架简介

Flask是轻量级Python Web框架（微框架），只提供核心功能，其他通过扩展添加。

特性FlaskDjangoFastAPI
定位微框架全栈框架现代API框架
学习难度低中低-中
异步支持有限部分原生支持
ORM无内置内置无内置
pip install flask

from flask import Flask
app = Flask(__name__)

@app.route('/')
def index():
    return 'Hello, Flask!'

if __name__ == '__main__':
    app.run(debug=True, port=5000)

3️⃣ 路由与视图函数

# 基本路由
@app.route('/about')
def about():
    return '关于我们'

# 动态路由
@app.route('/user/')
def user_profile(username):
    return f'用户: {username}'

# int类型参数
@app.route('/post/')
def show_post(post_id):
    return f'文章ID: {post_id}'

# 指定HTTP方法
@app.route('/login', methods=['GET', 'POST'])
def login():
    if request.method == 'POST':
        return '处理登录'
    return '显示登录表单'
转换器说明示例
string默认，不含斜杠/user/admin
int正整数/post/42
float浮点数/price/99.9
path含斜杠/files/a/b/c

4️⃣ 请求与响应对象

from flask import request, jsonify, redirect, url_for, abort

@app.route('/search')
def search():
    q = request.args.get('q', '')
    page = request.args.get('page', 1, type=int)
    return f'搜索: {q}, 页码: {page}'

@app.route('/api/user', methods=['POST'])
def create_user():
    data = request.get_json()
    return jsonify({'name': data.get('name')})

# 重定向
@app.route('/old')
def old():
    return redirect(url_for('index'))

# 错误处理
@app.errorhandler(404)
def not_found(e):
    return '页面不存在', 404

5️⃣ 模板渲染Jinja2

# templates/base.html
<!DOCTYPE html>
<html><head><title>{% block title %}我的网站{% endblock %}</title></head>
<body>
<nav><a href="{{ url_for('index') }}">首页</a></nav>
{% block content %}{% endblock %}
</body></html>

# templates/index.html
{% extends 'base.html' %}
{% block content %}
<h1>欢迎, {{ username }}!</h1>
{% if items %}
<ul>{% for item in items %}<li>{{ item }}</li>{% endfor %}</ul>
{% endif %}
{% endblock %}

# 视图函数
@app.route('/')
def index():
    return render_template('index.html',
        username='周同学', items=['Python','Flask','Web'])
语法说明
{{ var }}输出变量
{% if %}条件判断
{% for %}循环
{% extends %}模板继承
{{ var|filter }}过滤器

6️⃣ 实战：个人名片页

🎯 实战目标：使用Flask+Jinja2创建个人名片网页
from flask import Flask, render_template
app = Flask(__name__)

my_info = {
    'name': '周同学', 'title': 'Python开发工程师',
    'bio': '热爱编程，专注Python后端开发',
    'skills': ['Python','Flask','Django','MySQL','Docker'],
}
projects = [
    {'name':'个人博客','desc':'Flask博客平台'},
    {'name':'数据分析','desc':'Pandas数据分析'},
]

@app.route('/')
def index():
    return render_template('index.html', info=my_info)

@app.route('/projects')
def projects_page():
    return render_template('projects.html', projects=projects)

if __name__ == '__main__':
    app.run(debug=True)
