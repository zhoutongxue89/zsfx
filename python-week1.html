Day 1 · 认识Python与安装环境

1.1 什么是Python？

在开始安装之前，我们先来认识一下Python。Python是一种编程语言。你可以把它理解成一种和电脑对话的工具——你用Python写代码，电脑就能理解你的意图并执行相应操作。

Python的三大特点：
  • 简单易学 - 语法像说话一样自然，入门快，2周就能写小程序
  • 功能强大 - 什么都能做：Web开发、数据分析、AI、自动化...
  • 库超多 - 别人写好的工具可以直接用，不用重复造轮子

1.2 安装Python（Windows系统）

步骤1：确认你的电脑系统

  1. 按 Win + I 打开设置
  2. 点击"系统"
  3. 点击"关于"
  4. 查看"系统类型"
注意：
  • 如果显示"64位操作系统" → 安装64位Python（推荐）
  • 如果显示"32位操作系统" → 安装32位Python

步骤2：下载Python

访问：https://www.python.org/downloads/

点击黄色按钮 "Download Python 3.12.x"

步骤3：安装Python（关键步骤！）

⚠️ 这一步非常重要，很多人安装失败就是因为漏掉了勾选！
  1. 双击下载的 .exe 文件
  2. 一定要勾选 ☑ Add Python to PATH（添加到环境变量）
  3. 点击 "Install Now"（立即安装）

步骤4：验证安装成功

  1. 按 Win + R，输入 cmd，回车
  2. 输入：python --version
  3. 如果显示类似 Python 3.12.3，说明安装成功

1.3 安装VS Code

VS Code是微软开发的代码编辑器，是写代码的"笔记本"。

  1. 访问：https://code.visualstudio.com/
  2. 点击"Download for Windows"
  3. 安装Python插件：按 Ctrl + Shift + X，搜索"Python"，点击安装

1.4 第一个Hello World

  1. 打开VS Code，按 Ctrl + N 新建文件
  2. 按 Ctrl + S 保存为 hello.py
  3. 输入代码：
print("Hello World!")
print("你好，世界！")
  1. 按 Ctrl + Alt + N 运行（或右键选择"运行Python文件"）
🎉 恭喜你！你已经完成了第一个Python程序！

Day 3 · 变量基础

3.1 什么是变量？

变量就像一个带标签的盒子：

  • 变量名 = 标签（给数据起的名字）
  • 变量值 = 盒子里的东西（数据本身）

3.2 创建变量

# 基本语法：变量名 = 值
name = "张三"    # 创建变量name，值为"张三"
age = 25         # 创建变量age，值为25
height = 1.75    # 创建变量height，值为1.75
注意：等号 = 的含义是"赋值"，不是"等于"！

3.3 变量的命名规则

规则正确示例错误示例
由字母、数字、下划线组成name, age2, user_nameuser-name, my age
不能以数字开头score11score
区分大小写Name ≠ name-
不能用Python关键字-class, if

3.4 input函数

# 基本语法
name = input("请输入你的名字：")
print("你好，" + name)
重要：input() 返回的永远是字符串类型！

3.5 类型转换

# 转换为整数
age = int(input("年龄："))   # age = 18（整数）

# 转换为浮点数
height = float(input("身高："))  # height = 1.75

# 转换为字符串
num = str(123)                # num = "123"

Day 4 · print输出函数

4.1 f-string格式化（重点！）

f-string 是Python 3.6引入的格式化方法，语法简洁，功能强大。

name = "小明"
age = 18

# 普通拼接（繁琐）
print("我叫" + name + "，今年" + str(age) + "岁")

# f-string（推荐！）
print(f"我叫{name}，今年{age}岁")

4.2 数值格式化

格式说明示例结果
{value:.2f}保留2位小数f"{3.14159:.2f}"3.14
{value:,}千位分隔符f"{1234567:,}"1,234,567
{value:%}百分比格式f"{0.856:%}"85.6%

4.3 对齐格式化

name = "小明"
score = 95

print(f"|{name:<10}|")  # 左对齐 |小明        |
print(f"|{name:>10}|")  # 右对齐 |        小明|
print(f"|{name:^10}|")  # 居中 |     小明   |
print(f"{score:05}")    # 补零 00095

while True:
    try:
        age = int(input("年龄："))
        if 0 < age < 150:
            break
        else:
            print("年龄必须在1-149之间！")
    except ValueError:
        print("请输入有效数字！")

Day 7 · 运算符入门

7.1 算术运算符

a, b = 10, 3

print(a + b)    # 13  加法
print(a - b)    # 7   减法
print(a * b)    # 30  乘法
print(a / b)    # 3.33  除法（浮点数）
print(a // b)   # 3   整数除法
print(a % b)    # 1   取余
print(a ** b)   # 1000 幂运算

7.2 比较运算符

print(10 == 20)   # False  等于
print(10 != 20)   # True   不等于
print(10 > 20)    # False  大于
print(10 < 20)    # True   小于

# 链式比较（Python特色）
x = 5
print(0 < x < 10)      # True

7.3 逻辑运算符

print(True and False)   # False  与
print(True or False)    # True   或
print(not True)          # False  非

7.4 运算符优先级

优先级运算符说明
1()括号（最高）
2**幂运算
3*, /, //, %乘除
4+, -加减
5>, <, >=, <=比较
