1️⃣ FastAPI简介

FastAPI是现代高性能Web框架，基于Python类型注解，支持异步，自动生成文档。

优势说明
高性能媲美Node.js和Go
异步支持原生async/await
自动文档Swagger UI + ReDoc
类型安全基于Pydantic自动验证
pip install fastapi uvicorn[standard]

2️⃣ 快速入门

from fastapi import FastAPI
app = FastAPI(title="我的API", version="1.0.0")

@app.get("/")
async def read_root():
    return {"message": "Hello, FastAPI!"}

@app.get("/items/{item_id}")
async def read_item(item_id: int, q: str = None):
    return {"item_id": item_id, "q": q}

# 运行: uvicorn main:app --reload
# 文档: http://127.0.0.1:8000/docs

3️⃣ 路径与查询参数

# 路径参数（类型自动转换）
@app.get("/users/{user_id}")

    return {"user_id": user_id}

# 查询参数

    return {"skip": skip, "limit": limit}

from typing import Optional
@app.get("/search")
async def search(q: Optional[str] = None, active: bool = True):
    return {"q": q, "active": active}

4️⃣ Pydantic模型

from pydantic import BaseModel, Field, EmailStr
from typing import Optional, List

class UserCreate(BaseModel):
    username: str = Field(..., min_length=3, max_length=20)
    email: EmailStr
    password: str = Field(..., min_length=6)
    age: Optional[int] = Field(None, ge=0, le=150)
    tags: List[str] = []

class UserResponse(BaseModel):
    id: int
    username: str
    email: str
    
    class Config:
        from_attributes = True

@app.post("/users", response_model=UserResponse, status_code=201)
async def create_user(user: UserCreate):
    # FastAPI自动验证user数据
    return {"id": 1, "username": user.username, "email": user.email}
💡 数据不符合模型定义时，FastAPI自动返回422错误和详细错误信息。

5️⃣ 依赖注入

from fastapi import Depends

# 数据库会话依赖
def get_db():
    db = SessionLocal()
    try:
        yield db
    finally:
        db.close()

@app.get("/users/{uid}")
async def get_user(uid: int, db = Depends(get_db)):
    user = db.query(User).filter(User.id == uid).first()
    if not user:
        raise HTTPException(404, "用户不存在")
    return user

# 认证依赖
async def verify_token(x_token: str = Header(...)):
    if x_token != "secret":
        raise HTTPException(401, "无效token")

@app.get("/secure", dependencies=[Depends(verify_token)])
async def secure_data():
    return {"data": "protected"}

6️⃣ 异步与后台任务

import asyncio, httpx
from fastapi import BackgroundTasks

@app.get("/async-demo")
async def async_demo():
    async with httpx.AsyncClient() as client:
        tasks = [client.get(f"https://api.example.com/{i}") for i in range(5)]
        results = await asyncio.gather(*tasks)
    return {"count": len(results)}

def send_email(email, msg):
    print(f"发送到{email}: {msg}")

@app.post("/register")
async def register(email: str, bg: BackgroundTasks):
    bg.add_task(send_email, email, "欢迎注册！")
    return {"message": "注册成功"}
