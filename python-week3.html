1什么是条件判断

在现实生活中，我们每天都在做"如果...就..."的决策。程序也需要这种能力——根据不同的条件，执行不同的代码块。这就是条件判断。

💡

条件判断是程序控制流的基础。有了条件判断，程序就从"顺序执行"变成了能"思考"的智能代码。

开始 ↓ 条件是否成立？
✅ 是 ❌ 否
执行代码块A 执行代码块B ↓ 继续执行
🔑 布尔值
条件的结果是 True 或 False
📐 缩进
Python用4个空格缩进区分代码块
🔗 冒号
if/elif/else 后面必须加冒号 :
⚡ 短路求值
and/or 有短路特性，提高效率
2if 语句

最基本的条件语句，当条件为 True 时才执行对应代码块。

基本语法

# 基本语法结构
if 条件表达式:
    代码块  # 注意：必须缩进4个空格！

实际例子

# 例1：判断年龄是否成年
age = 18

if age >= 18:
    print("你已经成年了！")
    print("可以去投票了")

print("程序继续执行")  # 这行不受if影响
▶ 输出结果：
你已经成年了！
可以去投票了
程序继续执行 
# 例2：检查数字是否为正数
number = -5

if number > 0:
    print(f"{number} 是正数")

print("检查完毕")
▶ 输出结果（number=-5时）：
检查完毕
⚠️

注意缩进！ Python用缩进来区分代码块。if下面的代码必须缩进（通常4个空格），否则会报 IndentationError 错误。

✅ 正确写法
if age >= 18:
    print("成年")
❌ 错误写法（无缩进）
if age >= 18:
print("成年")  # 报错！
3if-else 语句

当条件成立时执行 if 块，不成立时执行 else 块，二选一。

基本语法

if 条件表达式:
    条件为 True 时执行
else:
    条件为 False 时执行

实际例子

# 例1：判断奇偶数
num = 7

if num % 2 == 0:
    print(f"{num} 是偶数")
else:
    print(f"{num} 是奇数")
▶ 输出结果：
7 是奇数 
# 例2：判断分数是否及格
score = 85

if score >= 60:
    print("及格了！恭喜🎉")
else:
    print("不及格，继续努力！")
▶ 输出结果：
及格了！恭喜🎉 
# 例3：用户登录判断
correct_password = "python123"
input_password = input("请输入密码：")

if input_password == correct_password:
    print("✅ 登录成功！欢迎回来")
else:
    print("❌ 密码错误，请重试")
4if-elif-else 多分支

当有多个条件分支时，用 elif（else if 的缩写）来处理多种情况。

💡

elif 可以有任意多个，Python 会从上到下依次检查，找到第一个满足的条件就执行对应代码块，然后跳出整个 if 结构。

基本语法

if 条件1:
    条件1为 True 时执行
elif 条件2:
    条件2为 True 时执行
elif 条件3:
    条件3为 True 时执行
else:
    以上都不满足时执行

成绩等级划分

# 经典的成绩分级
score = int(input("请输入成绩（0-100）："))

if score >= 90:
    grade = "A 优秀 🏆"
elif score >= 80:
    grade = "B 良好 👍"
elif score >= 70:
    grade = "C 中等 📚"
elif score >= 60:
    grade = "D 及格 ✅"
else:
    grade = "F 不及格 😢"

print(f"你的成绩：{score}分，等级：{grade}")
▶ 输入 85 的输出：
你的成绩：85分，等级：B 良好 👍

季节判断

# 根据月份判断季节
month = 8

if month in [3, 4, 5]:
    season = "春季 🌸"
elif month in [6, 7, 8]:
    season = "夏季 ☀️"
elif month in [9, 10, 11]:
    season = "秋季 🍂"
else:
    season = "冬季 ❄️"

print(f"{month}月是{season}")
▶ 输出结果：
8月是夏季 ☀️
5比较运算符

比较运算符用于构建条件表达式，结果总是 True 或 False。

运算符含义示例结果
==等于5 == 5True
!=不等于5 != 3True
>大于7 > 5True
<小于3 < 5True
>=大于等于5 >= 5True
<=小于等于4 <= 5True
in包含于"a" in "abc"True
not in不包含于"x" not in "abc"True
is是同一对象x is None取决于x
# 比较运算符示例
a = 10
b = 20
name = "Python"
fruits = ["苹果", "香蕉", "橙子"]

print(a == b)           # False
print(a != b)           # True
print(a < b)            # True
print("P" in name)     # True
print("苹果" in fruits)  # True
print("西瓜" not in fruits) # True

# 链式比较（Python特有）
age = 25
print(18 <= age <= 60)  # True，Python独特写法！
⚠️

== vs =：== 是比较运算符（判断相等），= 是赋值运算符（赋值）。初学者容易混淆！

6逻辑运算符

当需要组合多个条件时，使用逻辑运算符 and、or、not。

运算符含义说明示例
and且（与）两个条件都要为Truea > 0 and b > 0
or或至少一个条件为Truea > 0 or b > 0
not非（取反）对结果取反not (a > 0)

and（且）

# 判断是否满足工作条件（年龄在18-60岁之间）
age = 25

if age >= 18 and age <= 60:
    print("符合工作年龄要求")
else:
    print("不符合工作年龄要求")

# 更 Python 风的写法（链式比较）
if 18 <= age <= 60:
    print("符合要求（链式比较）")

or（或）

# 判断是否是周末
day = "Saturday"

if day == "Saturday" or day == "Sunday":
    print("今天是周末！放松一下🎉")
else:
    print("工作日，加油！")

# 更简洁的写法
if day in ["Saturday", "Sunday"]:
    print("今天是周末！")

not（非）

# 判断列表是否为空
items = []

if not items:
    print("购物车是空的！")
else:
    print(f"购物车有 {len(items)} 件商品")

# not 取反示例
is_raining = False
if not is_raining:
    print("不下雨，可以出门！")
✅

短路求值： and 如果第一个条件为False，就不检查第二个；or 如果第一个条件为True，就不检查第二个。这样可以避免错误。

# 短路求值保护 - 避免除以零错误
divisor = 0
value = 10

# 如果 divisor == 0，第二个条件不会执行，安全！
if divisor != 0 and value / divisor > 5:
    print("结果大于5")
else:
    print("除数不能为零")
7嵌套条件

if 语句可以嵌套在另一个 if 语句内部，用于处理更复杂的逻辑。

⚠️

嵌套不宜过深（一般不超过3层），太多嵌套会让代码难以阅读。如果嵌套过深，考虑用 and 合并条件或者拆分成函数。

# 例：购票资格判断
age = 25
has_ticket = True
is_vip = False

if age >= 18:
    print("年龄符合要求")
    if has_ticket:
        print("有票，可以进场")
        if is_vip:
            print("VIP通道 →")
        else:
            print("普通通道 →")
    else:
        print("没有票，无法进场")
else:
    print("年龄不符合，未成年人谢绝入场")
▶ 输出结果：
年龄符合要求
有票，可以进场
普通通道 →
❌ 避免过深嵌套
if a:
    if b:
        if c:
            if d:
                print("太深了")
✅ 用 and 合并条件
if a and b and c and d:
    print("更清晰！")
8三元表达式（条件表达式）

Python 提供了简洁的一行写法，适合简单的二选一情况。

语法格式

# 普通写法（if-else）
if 条件:
    结果 = 值A
else:
    结果 = 值B

# 三元表达式（一行搞定）
结果 = 值A if 条件 else 值B

实际示例

# 例1：判断奇偶
num = 7
result = "偶数" if num % 2 == 0 else "奇数"
print(result)  # 奇数

# 例2：取两个数中的最大值
a, b = 10, 20
max_val = a if a > b else b
print(f"最大值是：{max_val}")  # 最大值是：20

# 例3：显示状态
is_online = True
status = "在线 🟢" if is_online else "离线 🔴"
print(f"用户状态：{status}")  # 用户状态：在线 🟢

# 例4：带f-string的三元
score = 75
print(f"成绩{score}分，{'及格' if score >= 60 else '不及格'}")
💡

三元表达式适合简单的判断，复杂逻辑还是用多行 if-else 更清晰。不要为了"酷"而牺牲可读性。

9实战练习
🎯 练习题（从易到难）
  1. 【基础】 输入一个整数，判断是正数、负数还是零
  2. 【基础】 输入年份，判断是否为闰年（能被4整除且不被100整除，或能被400整除）
  3. 【中级】 制作一个简单的计算器：输入两个数和运算符（+、-、*、/），输出结果
  4. 【中级】 BMI计算器：输入身高（米）和体重（千克），计算BMI并给出健康建议
  5. 【挑战】 猜数字游戏：设定一个1-100的数字，用户输入猜测，提示"猜大了"、"猜小了"或"恭喜猜对！"

练习1：正负零判断

# 练习1：判断正负零
num = int(input("请输入一个整数："))

if num > 0:
    print(f"{num} 是正数 ➕")
elif num < 0:
    print(f"{num} 是负数 ➖")
else:
    print("这是零 ⭕")

练习2：闰年判断

# 练习2：闰年判断
year = int(input("请输入年份："))

# 闰年条件：(能被4整除 且 不能被100整除) 或 能被400整除
if (year % 4 == 0 and year % 100 != 0) or (year % 400 == 0):
    print(f"{year}年是闰年 🌙")
else:
    print(f"{year}年是平年 📅")

练习3：BMI计算器

# 练习3：BMI健康计算器
height = float(input("请输入身高（米）："))
weight = float(input("请输入体重（公斤）："))

bmi = weight / (height ** 2)
print(f"\n您的BMI指数：{bmi:.2f}")

if bmi < 18.5:
    print("体重过轻 😟 建议适当增加营养")
elif bmi < 24:
    print("体重正常 😊 继续保持健康生活方式")
elif bmi < 28:
    print("体重偏重 😐 建议适当运动")
else:
    print("肥胖 😟 建议咨询医生，调整饮食运动")
▶ 示例输出（身高1.75m，体重70kg）：
您的BMI指数：22.86
体重正常 😊 继续保持健康生活方式

综合挑战：猜数字游戏

# 挑战：猜数字游戏
secret = 42  # 假设答案是42

guess = int(input("猜猜我想的数字（1-100）："))

if guess < secret:
    print("猜小了 📉，再往大猜！")
elif guess > secret:
    print("猜大了 📈，再往小猜！")
else:
    print("🎉 恭喜你猜对了！答案就是42！")
