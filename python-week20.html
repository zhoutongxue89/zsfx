1️⃣ 为什么要测试

🔑 测试的价值
  • 确保代码正确性
  • 防止回归（改代码引入新bug）
  • 文档作用（测试即文档）
  • 增强重构信心
💡 测试金字塔
  • 单元测试：测试单个函数/方法（最多）
  • 集成测试：测试模块间协作
  • 端到端测试：测试整个系统（最少）

2️⃣ unittest框架

Python标准库提供的单元测试框架。

import unittest

def add(a, b):
    return a + b

class TestMath(unittest.TestCase):
    def test_add_positive(self):
        self.assertEqual(add(1, 2), 3)
    
    def test_add_negative(self):
        self.assertEqual(add(-1, -2), -3)
    
    def test_add_zero(self):
        self.assertEqual(add(0, 5), 5)
    
    def test_add_strings(self):
        self.assertEqual(add("Hello ", "World"), "Hello World")

if __name__ == '__main__':
    unittest.main()

3️⃣ pytest框架

更简洁、功能更强大的测试框架。

# 安装
pip install pytest

# test_example.py
import pytest

def add(a, b):
    return a + b

def multiply(a, b):
    return a * b

class TestMath:
    def test_add(self):
        assert add(1, 2) == 3
    
    def test_add_negative(self):
        assert add(-1, -2) == -3
    
    @pytest.mark.parametrize("a,b,expected", [
        (1, 2, 3),
        (0, 0, 0),
        (-1, 1, 0),
    ])
    def test_add_params(self, a, b, expected):
        assert add(a, b) == expected
    
    @pytest.fixture
    def sample_list(self):
        return [1, 2, 3]
    
    def test_multiply(self, sample_list):
        assert multiply(2, 3) == 6
💡 运行命令
  • pytest - 运行所有测试
  • pytest -v - 详细输出
  • pytest -k "test_name" - 只运行匹配的测试
  • pytest --tb=short - 简短错误信息

4️⃣ 断言与异常测试

import pytest

def divide(a, b):
    if b == 0:
        raise ValueError("除数不能为零")
    return a / b

class TestDivide:
    def test_normal(self):
        assert divide(10, 2) == 5
    
    def test_zero_division(self):
        with pytest.raises(ValueError, match="除数不能为零"):
            divide(10, 0)
    
    def test_float_division(self):
        result = divide(10, 3)
        assert abs(result - 3.333) < 0.001

# 使用pytest.raises测试异常
def test_exception():
    with pytest.raises(ZeroDivisionError):
        1 / 0

7️⃣ 调试技术

📌 print调试

# 最简单但有效
def buggy_function(data):
    result = []
    for i, item in enumerate(data):
        print(f"处理第{i}项: {item}")  # 调试打印
        result.append(item * 2)
    return result

📌 pdb调试器

import pdb

def buggy_function(data):
    pdb.set_trace()  # 设置断点
    result = []
    for item in data:
        result.append(item * 2)
    return result

# 常用命令：
# n (next) - 执行下一行
# s (step) - 进入函数
# c (continue) - 继续执行
# p variable - 打印变量
# l (list) - 显示代码
# q (quit) - 退出

📌 breakpoint() (Python 3.7+)

def buggy_function(data):
    breakpoint()  # 等价于 pdb.set_trace()
    return [x * 2 for x in data]

8️⃣ 日志记录

import logging

# 配置日志
logging.basicConfig(
    level=logging.DEBUG,
    format='%(asctime)s - %(name)s - %(levelname)s - %(message)s',
    handlers=[
        logging.FileHandler('app.log'),
        logging.StreamHandler()
    ]
)

logger = logging.getLogger(__name__)

def process_data(data):
    logger.debug(f"输入数据: {data}")
    try:
        result = data * 2
        logger.info(f"处理成功: {result}")
        return result
    except Exception as e:
        logger.error(f"处理失败: {e}")
        raise
